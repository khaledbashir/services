# ANC Service Dashboard — Overview

A unified operations platform managing events, staff, workflows, tickets, and client services across 43+ venues and 15 markets.

## What It Does

**Operations:** Auto-syncs 250+ events from Google Calendar. Calendar and list views. Assign techs to events with workload visibility. 3-stage mobile workflow: Check-in → Game Ready → Post-Game Report.

**Dashboard:** Real-time alerts surface problems — unstaffed events, missed check-ins, overdue reports. Labor budget tracking with estimated hours by league. Clickable stat cards, workflow charts, market breakdowns.

**Ticketing:** Full lifecycle with SLA tracking (auto-deadlines by priority), auto-assignment rules, internal/external comments, canned responses, and Slack notifications to venue channels.

**Staff:** Card and list views with profile photos. Detail pages showing hours, completion rate, markets covered. Bulk import via CSV or AI.

**Client Portal:** Unique link per venue, no login. Live game day tracking (like package tracking but for events). AI-powered ticket creation from plain English. Monthly branded PDF service reports. Installed display specs auto-populated from the Proposal Engine.

**Reports:** Executive-grade PDF reports with trend arrows, SLA compliance, coverage by market, and action items. Built for forwarding to leadership.

**Integrations:** Google Calendar, Slack (AI bot + notifications), Proposal Engine (auto-creates venues when deals close), branded PDF generation.

**Security:** Role-based access — Admin, Manager, Technician. Every API enforced server-side. Client portal fully isolated.
